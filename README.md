### Prerequisites

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Senayldz/Acne-Detect-App.git
cd Acne-Detect-App
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **iOS Setup** (iOS only)
```bash
cd ios
bundle install
bundle exec pod install
cd ..
```

4. **Start Metro server**
```bash
npm start
# or
yarn start
```

5. **Run the app**
```bash
# Android
npm run android

# iOS
npm run ios
```
